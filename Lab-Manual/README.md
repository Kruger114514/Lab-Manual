# 实验室日常清洁流程 

## 垃圾处理 
每周四需要记得在系统上下单，然后在次日（周五）中午11时左右将处理后的黄色垃圾袋放置于实验室门口。下单填写实例如下： 

![回收单填写示例](下单界面操作指南.png) 

![下单成功后显示](最终结果.png) 

黄色垃圾袋处理之后，需要使用轧带封口，并贴上危废品警告标签，按照以下要求填写标签。最后在垃圾袋上写上本组名字。 

![垃圾袋标签填写](垃圾袋标签填写.png)
